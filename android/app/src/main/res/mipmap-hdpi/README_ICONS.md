# App Icons Placeholder

This directory should contain the app launcher icons for different screen densities:

- ic_launcher.png (48x48dp) - Place in each mipmap directory
- Recommended icon sizes:
  - mipmap-mdpi: 48x48px
  - mipmap-hdpi: 72x72px  
  - mipmap-xhdpi: 96x96px
  - mipmap-xxhdpi: 144x144px
  - mipmap-xxxhdpi: 192x192px

To generate app icons easily, you can use:
1. Android Studio's Image Asset Studio
2. Online tools like https://romannurik.github.io/AndroidAssetStudio/
3. Flutter's flutter_launcher_icons package

For now, the app will use the default Flutter launcher icon.
